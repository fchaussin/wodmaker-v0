Enhanced Numeric Input

Touch-optimized numeric input with fullscreen modal interface

How to Use

  • Tap any number input to open the fullscreen numeric keypad
  • Use + and - buttons to increment/decrement values
  • Hold buttons for accelerated value changes
  • Tap the number to use your device's native keypad
  • Keyboard shortcuts: Arrow keys, Enter to save, Escape to cancel
  • Save, Cancel, or Reset using the action buttons
